# **Television Ecija:**
*`"La television de Ecija tiene videos, fotos y diferente contenido almacenados de manera distribuida, en diferentes equipos, discos duros, etc. La idea es crear un software capaz de administrar, organizar, permitir busquedas y filtros y categorizar"`*

## **Requisitos:**
- Web local a nivel de empresa para administrar contenido.
- Subir y cargar archivos (Sobre todo Multimedia).
- Filtrar, ordenar y centralizar videos, imagenes.
- Permitir busqueda avanzada.
- Acceso a la aplicación mediante login.
- Permitir registrarse para solo visualizar contenido.

## **Stack (Tecnologias):**
- MySQL (Base de datos)
- Express NodeJS (Api)
- Angular (Frontend)

## **Dockerizar:**
- Creación de contenedores independientes
- Creación de volúmenes para la perpetuación de datos

## **Diseño:**
- Panel de busqueda
- Filtrado por separado y global
- Ordenacion por ambiguedad
- Control de acceso
- Creación, edición y borrado de usuarios
